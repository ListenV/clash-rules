## 特点

- 引用别人规则+自己平时收集的规则

## 说明

- `direct.yaml` 文件是默认`DIRECT`，匹配不到规则就直通，节省流量
- `rules.yaml` 旧版文件，可以进一步修改为大而全的「漏网之鱼」模式
- `collection/dns-pollution.list`：是针对SmartDNS的DNS污染域名列表
